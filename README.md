# مشروع Absher - Text to Speech

مشروع React Native مع Expo لتحويل النص إلى كلام.

## المتطلبات

- Node.js (الإصدار 14 أو أحدث)
- npm أو yarn
- Expo CLI
- تطبيق Expo Go على هاتفك (للتجربة على الجهاز)

## خطوات التشغيل

### 1. تثبيت التبعيات

```bash
npm install
```

أو

```bash
yarn install
```

### 2. تشغيل المشروع

```bash
npm start
```

أو

```bash
yarn start
```

### 3. فتح التطبيق

بعد تشغيل الأمر، ستظهر لك قائمة QR Code. يمكنك:

- **على Android**: اضغط `a` في Terminal أو افتح تطبيق Expo Go وامسح QR Code
- **على iOS**: اضغط `i` في Terminal أو افتح تطبيق Expo Go وامسح QR Code
- **على المتصفح**: اضغط `w` لفتحه في المتصفح

## الأوامر المتاحة

- `npm start` - تشغيل Expo
- `npm run android` - تشغيل على Android مباشرة
- `npm run ios` - تشغيل على iOS مباشرة
- `npm run web` - تشغيل على المتصفح
